   NOW:     Invoice appears immediately after clicking "Generate"! ✨

──────────────────────────────────────────────────────────────────────────
   NEW:     Red trash icon (🗑️) button for each invoice
   SMART:   Only shows when there are 2+ invoices
   SAFE:    Can't delete the last invoice (minimum 1 required)

═══════════════════════════════════════════════════════════════════════════
  🧪 HOW TO TEST
═══════════════════════════════════════════════════════════════════════════

Test 1: Auto-Refresh
──────────────────────────────────────────────────────────────────────────
  1. Go to any order details page
  2. Click "Generate" button
  3. ✨ Invoice appears immediately in the list!
     (No need to exit/refresh the page!)

Test 2: Delete Invoice
──────────────────────────────────────────────────────────────────────────
  1. Generate 2+ invoices for an order
  2. See red trash icon (🗑️) button for each invoice
  3. Click trash icon
  4. ✨ Invoice deleted + list updates immediately!
  5. See "Invoice deleted successfully" notification

Test 3: Protection
──────────────────────────────────────────────────────────────────────────
  1. Generate only 1 invoice
  2. ✨ No delete button appears
     (Can't delete the only invoice!)
