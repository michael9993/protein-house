# Saleor Platform - Complete Development Environment

A complete, production-ready Saleor e-commerce platform setup with API, Dashboard, Storefront, and Stripe payment integration.

## 🚀 Features

- **Saleor API** - Django/GraphQL backend with PostgreSQL
- **Saleor Dashboard** - Admin interface for managing your store
- **Saleor Storefront** - Customer-facing Next.js storefront
- **Stripe Payment App** - Official Stripe payment gateway integration
- **Docker Compose** - Complete local development environment
- **Tunnel Support** - Cloudflare/ngrok tunnel integration for webhooks
- **PostgreSQL** - All data stored in PostgreSQL (no DynamoDB required)

## 📋 Prerequisites

- Docker and Docker Compose
- PowerShell (for Windows setup scripts)
- Stripe account (for payment processing)
- Cloudflare CLI or ngrok (optional, for tunnels)

## 🏃 Quick Start

### 1. Clone and Setup

```powershell
# Navigate to infra directory
cd infra

# Run setup script
.\scripts\setup-environment.ps1 -WithTunnels
```

### 2. Start Services

```powershell
docker compose -f docker-compose.dev.yml up -d
```

### 3. Access Services

- **Saleor API**: http://localhost:8000/graphql/
- **Dashboard**: http://localhost:9000
- **Storefront**: http://localhost:3000
- **Stripe App**: http://localhost:3002

## 🔧 Configuration

All configuration is managed through:
- `infra/.env` - Environment variables (generated by setup script)
- `infra/docker-compose.dev.yml` - Docker services configuration

## 🔐 Security

- Never commit `.env` files
- Use environment variables for secrets
- Stripe keys are stored securely in the Stripe app configuration
