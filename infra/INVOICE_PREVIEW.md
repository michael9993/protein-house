## Color Palette Used

### PDF Invoice:

- **Primary Blue**: #2563EB (headers, totals box, accents)
- **Text Dark**: #1F2937 (main text)
- **Light Gray**: #F3F4F6 (alternating rows, boxes)
- **Border**: #E5E7EB (lines, separators)
- **White**: #FFFFFF (background)

### Email:

- **Header Blue**: #2563EB (gradient effect)
- **Button Blue**: #2563EB (call-to-action)
- **Text**: #374151 (main content)
- **Light Text**: #6B7280 (secondary info)
- **Very Light**: #F9FAFB (boxes, backgrounds)

## Professional Typography

- **Headings**: Helvetica-Bold, 32px → 12px (hierarchical)
- **Body Text**: Helvetica, 10px → 12px
- **Emphasis**: Bold weights for important info
- **Spacing**: 1.6 line-height for readability

## Mobile Responsive (Email)

The email automatically adapts:

**Desktop:**

- Full width layout
- Side-by-side sections
- Large buttons

**Mobile:**

- Stacked layout
- Full-width buttons
- Optimized spacing
- Touch-friendly targets
