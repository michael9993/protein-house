# Product Name Cleanup Guide

**How to use:** Go to Saleor Dashboard > Products > click each product > edit the name.
Copy the "Suggested Name" column. Names should be **under 60 characters**, natural language, no keyword stuffing.

**Priority:** Fix the "Bad" ones first (longest/most stuffed), then "OK" ones. "Good" names can stay.

## How to Batch Update

The fastest way is through the **Bulk Manager** app:

1. Go to Dashboard > Apps > Bulk Manager
2. Export Products as CSV
3. Edit the `name` column in a spreadsheet using this guide
4. Re-import the CSV with **Upsert Mode** enabled

This lets you rename all products in one go instead of clicking through each one individually.
